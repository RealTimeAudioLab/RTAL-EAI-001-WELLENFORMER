# Contributing

This repository is primarily maintained as an **Engineering Heritage Archive**.

Contributions are welcome when they help preserve, document or clarify the original project.

Recommended contribution types:

- documentation improvements,
- build notes for historical toolchains,
- verified compatibility notes,
- schematic or hardware documentation,
- non-invasive bug reports.

Please avoid silently modernizing the original source. If a port to a newer ESP32 Arduino Core is created, it should live in a separate branch or clearly marked folder.
