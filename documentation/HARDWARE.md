# Hardware Notes

This document records the known hardware context for the WELLENFORMER project.

## Platform

- ESP32 Dev Module
- No PSRAM required in the original build configuration
- SD card support
- SPI display / touchscreen user interface
- I2S audio output

## Touchscreen

The project includes the XPT2046 touchscreen support files:

- `XPT2046_Touchscreen.cpp`
- `XPT2046_Touchscreen.h`

These files retain their original MIT license notice.
